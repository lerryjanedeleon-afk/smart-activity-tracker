# 📊 Smart Daily Activity Tracker with Data Analysis and Visualization

## 📌 Description

The **Smart Daily Activity Tracker** is a Python-based program that helps users record, monitor, and analyze their daily activities. Users can input activities along with the number of hours spent, and the system processes this data to provide useful insights.

The program computes the **total time** and **average time** spent on activities and gives simple suggestions to improve productivity. It also includes a **histogram visualization** to display the distribution of activity hours.

---

## 🎯 Objectives

* To collect daily activity data from users
* To analyze data using basic statistical methods (total and average)
* To visualize data using a histogram
* To provide suggestions based on user activity

---

## ⚙️ Features

* Add daily activities with time spent
* Save and load data using a file (`activities.txt`)
* Display all recorded activities
* Compute total and average hours (output analysis)
* Provide simple productivity suggestions
* Show histogram for data visualization

---

## 🧠 Concepts Applied

* **Input Data Modeling** – collecting activity data
* **Output Analysis** – computing total and average (mean)
* **Data Visualization** – using histogram to show distribution
* **Quantitative Data** – numerical data (hours spent)
